# 📊 Shark Tank India – Exploratory Data Analysis (EDA)

## 📌 Project Overview
This project performs an in-depth Exploratory Data Analysis (EDA) on the dataset of the TV show Shark Tank India (Season 1) to analyze:

Investment trends

Shark-wise funding patterns

Equity distribution

Deal outcomes

Multi-shark collaboration behavior

## 🎯 Objectives

Clean and preprocess raw dataset

Handle missing values and inconsistencies

Analyze total investment distribution

Study individual shark investment behavior

Identify outliers in equity and deal valuation

Visualize funding patterns effectively

## 🛠 Tech Stack

Python

Pandas

NumPy

Matplotlib

Seaborn

Jupyter Notebook

## 🔎 Key Analysis Performed
### 1️⃣ Data Cleaning & Preprocessing

Removed duplicates

Treated missing values

Corrected data types

Standardized categorical variables

### 2️⃣ Investment Trend Analysis

Total funding amount

Highest funded startups

Average deal size

Industry-wise distribution

### 3️⃣ Shark-wise Analysis

Total investment per shark

Number of deals participated

Solo vs collaborative deals

Equity percentage trends

### 4️⃣ Equity & Valuation Study

Equity vs funding correlation

Outlier detection

Distribution plots & boxplots

## 📈 Sample Visualizations

Bar Charts (Investment comparison)

Box Plots (Equity distribution)

Correlation Heatmaps

Multi-shark deal graphs

## 📊 Insights Extracted

Identified most aggressive investor

Observed multi-shark partnership patterns

Found sectors receiving highest funding

Analyzed valuation-to-equity relationship
